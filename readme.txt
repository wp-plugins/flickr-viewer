Provides simple drag & drop image gallery functionality to enable you to display Flickr Photo Sets in your WordPress installation.


== Description ==

Flickr Photo Sets in your WordPress installation simply with Flickr Viewer. 

Display Photo sets

Display Random image from Photo set

Light box style gallery

Includes, Widgets & Shortcodes


== Prerequisites ==

1. PHP5
2. Apache Web Server
3. Flickr Account


== Installation ==

1. Unzip into your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Settings->Flickr to Authorize your Flickr account
4. Click 'Start the Login Process'
4. Authorize your Flickr Account
5. Place shortcode into page / post


== Shortcodes ==

*[cws_flickr_recent] Will display recent public photos. 
*[cws_flickr_photoset photoset_id='72157631841791311'] Will display photo set with ID '72157631841791311'
*[cws_flickr_photo_random photoset_id='72157631841791311'] Will display a random photo from given photo set with ID '72157631841791311'
